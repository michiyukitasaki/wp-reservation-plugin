A WordPress plugin to manage reservations with customizable settings and a user-friendly interface.

== Description ==

Easy Reservation Systems is a powerful and user-friendly WordPress plugin that allows you to manage reservations and bookings with ease. Customize your reservation settings, display a calendar for users to select dates, and handle reservations efficiently.

= Features =
* Customizable reservation settings
* User-friendly calendar interface
* Shortcode support for easy display
* Efficient reservation management tools

== Installation ==

1. Download the plugin file and upload it to the `/wp-content/plugins/` directory of your WordPress site.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to 'Reservation Settings' to customize your reservation options.

== Frequently Asked Questions ==

= How do I display the reservation form? =
Use the `[reservation_system]` shortcode on any page or post where you want the form to appear.

= Can I customize the available time slots? =
Yes, you can configure the available days, time slots, and maximum people per slot in the 'Reservation Settings' menu.
